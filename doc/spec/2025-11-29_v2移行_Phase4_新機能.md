### 2. シフト申請フロー実装

#### 管理者側

1. **シフト募集作成**
   - 期間を設定（例: 12/1〜12/7）
   - 作成時にスタッフ全員にマジックリンク発行
   - メール送信（シフト申請依頼）

2. **申請状況確認**
   - 誰が申請済みか一覧表示
   - 未申請者へのリマインド機能

3. **AIシフト作成**
   - 申請データ + スキル + 労働時間上限 + 必要人員 を入力
   - AIがシフト案を生成
   - 管理者が調整

4. **シフト確定**
   - 確定ボタンで全スタッフに通知
   - マジックリンク無効化

#### スタッフ側

1. **マジックリンクでアクセス**
   - メールのリンクをクリック
   - ログイン不要でシフト申請画面へ

2. **シフト希望入力**
   - カレンダーUIで日付選択
   - 時間入力
   - 送信

3. **確定シフト確認**
   - 確定後、メールで通知
   - リンクから確定シフトを閲覧

### 4. 将来機能（ヘルプ要請）

#### 概要

直前キャンセル時、同時間帯に希望を出していたスタッフへ連絡する機能。

#### フロー案

1. スタッフがキャンセル申請
2. システムが同時間帯の希望者を検索
3. 該当スタッフにヘルプ要請メール送信
4. スタッフが承諾 → シフト自動更新
