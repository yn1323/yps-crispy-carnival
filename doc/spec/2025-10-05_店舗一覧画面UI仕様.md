# 店舗一覧画面 UI仕様

**作成日**: 2025-10-05

## 概要

店舗の基本情報を一覧表示し、詳細画面や編集画面への導線を提供する画面。
カード形式でスマホ対応を重視した設計とする。

## 要件

### 表示情報

各店舗カードには以下の情報を表示する：

- **店舗名**: 店舗の名称
- **営業時間**: 開店時間〜閉店時間（例：9:00 - 22:00）
- **シフト提出頻度**: 頻度 + 時間単位（例：週1回 | 15分単位）
- **タイムカード機能**: ON/OFF状態

### UI形式

**カード形式**でレスポンシブ対応（PC/SP）

#### カードレイアウト

```
┌─────────────────────────┐
│ 🏪 新宿店       [編集]  │
│ ⏰ 9:00 - 22:00        │
│ 📅 週1回 | 15分単位    │
│ 🎫 タイムカード ON     │
└─────────────────────────┘
```

- PCでは複数列のグリッドレイアウト
- SPでは1列表示

### インタラクション

#### カード全体クリック
- 店舗詳細画面へ遷移
- 詳細画面ではスタッフ情報などを表示（別途実装予定）

#### [編集]ボタンクリック
- 店舗編集画面へ遷移（別途実装予定）
- カードクリックイベントの伝播を停止

### 空状態

店舗が1件も登録されていない場合の表示：

```
┌─────────────────────────┐
│                         │
│   📭 店舗がありません    │
│                         │
│   [+ 店舗を登録する]    │
│                         │
└─────────────────────────┘
```

- 「+ 店舗を登録する」ボタンクリック → 店舗登録画面へ遷移

## 技術詳細

### 使用技術
- Chakra UI カード形式コンポーネント
- グリッドレイアウト（SimpleGrid推奨）
- レスポンシブブレイクポイント（2段階：PC/SP）

### レイアウト例
```tsx
<SimpleGrid columns={[1, 2, 3]} gap={4}>
  {/* カードコンポーネント */}
</SimpleGrid>
```

### データ取得
- サーバーサイドでの店舗一覧取得
- 営業時間、シフト設定、タイムカード設定を含む
