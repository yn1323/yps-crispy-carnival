## 2. 変更対象

### スキーマ（convex/schema.ts）

#### shopUserBelongings → staffs

```typescript
// Before
const shopUserBelongings = defineTable({
  shopId: v.id("shops"),
  userId: v.id("users"),          // 削除
  displayName: v.string(),
  role: v.string(),               // 削除
  status: v.string(),
  inviteToken: v.optional(v.string()),
  inviteExpiresAt: v.optional(v.number()),
  invitedBy: v.optional(v.id("users")),
  resignedAt: v.optional(v.number()),
  resignationReason: v.optional(v.string()),
  memo: v.optional(v.string()),
  workStyleNote: v.optional(v.string()),
  maxWorkingHoursPerMonth: v.optional(v.number()),  // → maxWeeklyHours
  hourlyWage: v.optional(v.number()),
  createdAt: v.number(),
  isDeleted: v.boolean(),
});

// After
const staffs = defineTable({
  shopId: v.id("shops"),
  email: v.string(),              // 追加
  displayName: v.string(),
  status: v.string(),             // "pending" | "active" | "resigned"
  skills: v.optional(v.array(v.object({  // 追加
    position: v.string(),
    level: v.string(),
  }))),
  maxWeeklyHours: v.optional(v.number()),  // 追加（月→週）
  magicLinkToken: v.optional(v.string()),
  magicLinkExpiresAt: v.optional(v.number()),
  invitedBy: v.optional(v.string()),  // authIdに変更
  resignedAt: v.optional(v.number()),
  resignationReason: v.optional(v.string()),
  memo: v.optional(v.string()),
  workStyleNote: v.optional(v.string()),
  hourlyWage: v.optional(v.number()),
  createdAt: v.number(),
  isDeleted: v.boolean(),
});
```

#### shops テーブル

```typescript
// 削除するフィールド
useTimeCard: v.boolean(),  // 削除
```

#### users テーブル

- 管理者のみが使用
- 構造変更なし

// 削除
