## 1. 実装スコープ

### 対象画面（管理者向け）
| 画面 | 状態 | 説明 |

```
┌─────────────────────────────────────────────────┐
│  ← 戻る              シフト募集作成              │

│  │ │ [____]    │ │ [____]    │              │  │
│  │ └───────────┘ └───────────┘              │  │
│  └───────────────────────────────────────────┘  │
│  ┌───────────────────────────────────────────┐  │
│  │ ⚙️ 申請設定                               │  │
│  │ ┌───────────────────────────────────────┐ │  │
│  │ │ 申請締切日                             │ │  │
│  │ │ [____]                                 │ │  │

│                                                 │
```

- FormCard（2セクション: 募集期間、申請設定）

## 4. モックデータ定義

### 4.1 募集データ（既存拡張）

```typescript
type RecruitmentType = {
  _id: string;
  shopId: string;
  startDate: string;        // "2025-12-01"
  endDate: string;          // "2025-12-07"
  deadline: string;         // "2025-11-25"
  status: "open" | "closed" | "confirmed";
  appliedCount: number;
  totalStaffCount: number;
  createdAt: number;
  closedAt?: number;
  confirmedAt?: number;
};
```

### 4.2 申請データ（新規）

```typescript
type ShiftRequestType = {
  _id: string;
  recruitmentId: string;
  staffId: string;
  staffName: string;
  date: string;             // "2025-12-01"
  startTime: string;        // "09:00"
  endTime: string;          // "17:00"
  note?: string;
  createdAt: number;
};
```

### 4.3 確定シフトデータ（新規）

```typescript
type ConfirmedShiftType = {
  _id: string;
  recruitmentId: string;
  staffId: string;
  staffName: string;
  date: string;
  startTime: string;
  endTime: string;
  position?: string;
  createdAt: number;
};
```

### 4.4 必要人員データ（新規）

```typescript
type RequiredStaffingType = {
  _id: string;
```
