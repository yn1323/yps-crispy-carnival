# シフト俯瞰ビュー切替 実装計画

**作成日**: 2026-02-07
**目的**: 既存ShiftOverviewを拡張し、日毎ビュー（ShiftTableTest）と俯瞰ビュー（ShiftOverview）の切替を導入する

---

## 1. 背景

- 現状は日毎タブ切替（DateTabs）で1日分のシフトのみ表示
- `src/components/features/Shift/ShiftOverview/` に俯瞰ビューの基本実装が既に存在
- 2つのビューを切り替えるUIが未実装

## 2. 確定方針

| 項目 | 決定 |
|------|------|
| セル表示 | `09:00-17:00`（分まで含む）→ セル幅70→90px |
| 充足度 | フッターのサマリー行のみ（%+出勤数） |
| SP対応 | スコープ外 |
| ビュー切替 | SegmentGroup（日毎 / 俯瞰） |
| データ同期 | リアルタイム反映（親でuseUndoRedo管理、両ビュー共有） |
| props統一 | 日毎ビューの詳細データ構造をベースに統一 |
| UI状態保持 | CSS display:none で両コンポーネント常時マウント |

## 7. データフロー図

```
ShiftViewSwitcher (親)
├── useUndoRedo(initialShifts) → { shifts, setShifts, undo, redo }
├── useState: viewMode ("daily" | "overview")
├── useState: selectedDate
├── useKeyboardShortcuts (viewMode === "daily" のみ)
│
├── [display:block/none] ShiftTableTest
│     受取: shifts, onShiftsChange, selectedDate, onDateChange, undo/redo
│     内部保持: toolMode, sortMode, popover, drag, scroll（display:none で保持）
│     編集操作 → onShiftsChange → 親の shifts 更新
│
└── [display:block/none] ShiftOverview
      受取: shifts（同じ参照）, dates, onDateClick
      内部保持: sortMode, selectedStaffId（display:none で保持）
      日付クリック → setSelectedDate + setViewMode("daily")
```

**リアルタイム同期の仕組み**:
1. 日毎で編集（paint/erase/resize）
2. `setShiftsNormalized` → `onShiftsChange(normalizedShifts)` 呼び出し
3. 親の `useUndoRedo.set()` → `history.present` 更新
4. 俯瞰ビューも同じ `shifts` を props で参照
5. 切替時に `prepareStaffRowData(staffs, shifts, ...)` が useMemo で再計算
6. 俯瞰のセル表示（`09:00-17:00`）が最新状態を反映

**display:none のメリット**:
- 両コンポーネント常時マウント → React stateが消えない
- 切替時にマウント/アンマウントのコストなし（即座に表示）
- 非表示のコンポーネントでも useMemo は shifts 変更時に再計算されるが、DOMレイアウト/ペイントは発生しない

1. **selectedDate の範囲外**: 俯瞰の onDateClick は dates 配列内の日付のみ。DateTabs は該当タブを選択
2. **StaffEditModal の重複**: 両ビューで独立管理。display:none により一方のみ表示されるため問題なし
3. **useKeyboardShortcuts の再登録**: viewMode 変更時にハンドラーが変わるため effect が再実行。cleanup → 再登録で正常動作
4. **sortedStaffIds の初期化**: controlled 化後、`shifts` (props) と `selectedDate` (props) を使って初期ソート
5. **デバッグ情報セクション**: ShiftTableTest 内に残す。shifts は props 経由で表示
