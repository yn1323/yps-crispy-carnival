# SP版シフトコンポーネント簡素化計画

**作成日**: 2026-02-08
**目的**: SP版シフトコンポーネントをモバイルユースケースに最適化し、不要な機能を削除・簡素化する

---

## 1. 背景・方針

SPの実際のユースケース（出勤者確認・簡易編集・シフト追加）に対して現在の実装はオーバースペック。以下を簡素化する:

| 項目 | 変更前 | 変更後 |
|------|--------|--------|
| スタッフ表示 | 全スタッフ（未提出含む） | 出勤者のみ + 追加ボタン |
| 充足度（日毎） | カラーバー + 出勤/必要テキスト | 「出勤 X名」テキストのみ |
| 充足度（俯瞰） | カラーバー + 充足% + 必要数 | 「出勤 X名」テキストのみ |
| ポジション識別 | 色バーのみ | 色バー + ポジション名テキスト併記 |
| 俯瞰の未提出者 | ⚠アイコン付きリスト | 非表示 |
| ソート | 維持 | 維持 |
| Undo/Redo | 維持 | 維持 |

## 3. 実装ステップ

### Step 1: FulfillmentBar 簡素化
**ファイル**: `src/components/features/Shift/ShiftDailyCardSP/FulfillmentBar.tsx`

- 充足度カラーバーのパスを削除（requiredStaffing関連のロジック全削除）
- 常に「出勤 X名」テキストのみ表示（現在の `!hasRequired` パスと同等）
- `FILL_RATE_COLORS`、`getDailyRequiredCount`、`getFillRateColor` を削除
- Props から `requiredStaffing` を削除

### Step 2: StaffCard にポジションテキスト追加
**ファイル**: `src/components/features/Shift/ShiftDailyCardSP/StaffCard.tsx`

- MiniShiftBar の下にポジション名+時間テキスト行を追加
- 各セグメント（休憩除く）を `●{色ドット} {ポジション名} {開始}-{終了}` で表示
- 例: `●ホール 09:00-12:00  ●キッチン 13:00-17:00`

### Step 3: index.tsx - 出勤者のみ表示 + 追加ボタン
**ファイル**: `src/components/features/Shift/ShiftDailyCardSP/index.tsx`

- `staffs.map(...)` を出勤者のみにフィルタ（当日の `positions.length > 0` のシフトがあるスタッフ）
- カードリスト下部に「+ スタッフを追加」ボタンを配置
- ボタンタップ → StaffAddSheet（新規）を開く
- StaffAddSheet で選択したスタッフ → ShiftEditSheet に遷移

### Step 4: StaffAddSheet 新規作成
**ファイル**: `src/components/features/Shift/ShiftDailyCardSP/StaffAddSheet.tsx`（新規）

- BottomSheet で未出勤スタッフ一覧を表示
- 各行: スタッフ名 + ステータス（未提出 / 希望時間 / 希望なし）
- タップ → onSelectStaff コールバック（親で ShiftEditSheet を開く）
- 既存 `BottomSheet` UI を使用

### Step 5: DateCard 簡素化
**ファイル**: `src/components/features/Shift/ShiftOverviewCardSP/DateCard.tsx`

- 充足度%テキスト削除（ヘッダー右側の「充足 80%」）
- 充足度カラーバー削除
- 未提出スタッフリスト削除（`unsubmittedStaffs.map(...)` 部分）
- フッターを「出勤 X名」のみに（「/ 必要 Y名」を削除）
- `requiredCount` Prop を削除
- `FILL_RATE_COLORS` import 削除

### Step 6: ShiftOverviewCardSP index 簡素化
**ファイル**: `src/components/features/Shift/ShiftOverviewCardSP/index.tsx`

- `getDailyRequiredCount` 削除
- `requiredCount` の計算・DateCard への受け渡し削除

### Step 7: 型定義更新
**ファイル**:
- `src/components/features/Shift/ShiftDailyCardSP/types.ts`
  - `FulfillmentBarProps` から `requiredStaffing` 削除
  - `StaffAddSheetProps` 型を追加
- `src/components/features/Shift/ShiftOverviewCardSP/types.ts`
  - `DateCardProps` から `requiredCount` 削除

### Step 8: Storybook 更新
**ファイル**:
- `src/components/features/Shift/ShiftDailyCardSP/index.stories.tsx`
- `src/components/features/Shift/ShiftOverviewCardSP/index.stories.tsx`
