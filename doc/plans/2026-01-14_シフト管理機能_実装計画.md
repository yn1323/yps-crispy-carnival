# シフト管理機能 - 実装計画

**作成日**: 2026-01-14
**ステータス**: 作業中
**参照仕様書**: [2026-01-03_シフト管理機能.md](../spec/2026-01-03_シフト管理機能.md)

---

## 1. 概要

管理者画面のシフト管理機能をモック実装する。
スタッフのシフト希望を収集し、管理者が確定するワークフローを提供する。

---

## 2. スコープ

### 対象

| 画面/機能 | 状態 | 説明 |
|-----------|------|------|
| 募集一覧 | 既存 | RecruitmentList 実装済み |
| 募集作成 | 新規 | シンプルなフォーム |
| 申請状況確認 | 新規 | スタッフ申請状況の確認 |
| シフト確定 | 新規 | 希望ベースの手動調整 |
| 必要人員設定 | 新規 | 曜日×時間帯×ポジション |

### 対象外（今回のスコープ外）

- スタッフ用画面（マジックリンク経由）
- DBスキーマ（Convex schema）
- API実装（mutations/queries）
- メール送信（Resend連携）

---

## 3. 技術的な決定事項

- **UIパターン**: 既存の ShopForm、StaffList パターンを踏襲
- **フォーム**: React Hook Form + Zod バリデーション
- **レスポンシブ**: モバイルファースト、PC/SP で表示切り替え
- **コンポーネント配置**: Chakra UI v3 Modern API 準拠

---
