# シフト俯瞰ビュー（ShiftOverview）仕様書

**作成日**: 2026-01-29
**目的**: シフト提出期間全体を俯瞰し、最終チェックを行うためのビュー

---

## 1. 概要

管理者がシフト提出期間（2週間など）全体を一覧で確認するためのテーブルコンポーネント。
各スタッフの勤務時間と月別労働時間を俯瞰し、問題箇所を発見して1日ビューへ遷移する。

### 目的・役割

| 項目 | 内容 |
|------|------|
| **主目的** | シフトを1枚で見やすくして最終チェック |
| **副目的** | 労働時間の過不足アラート確認 |
| **役割分担** | 俯瞰→問題発見→1日ビューで詳細編集 |

### ファイル構成（予定）

## 2. 機能一覧

### 2.1 テーブル表示

#### 基本構造

```
         │ 1/27(月) │ 1/28(火) │ ... │ 2/8(土) │ 2/9(日) ││ 1月計 │ 2月計 │
─────────┼──────────┼──────────┼─────┼─────────┼─────────┼───────┼───────┤
田中太郎 │ 9-17     │ -        │ ... │ 10-18   │ -       ││ 32h   │ 16h   │
山田花子 │ -        │ 11-19    │ ... │ -       │ 9-15    ││ 24h ⚠ │ 12h   │
鈴木一郎 │ -        │ -        │ ... │ -       │ -       ││ 0h    │ 0h    │
```

#### セル内容

| セル位置 | 表示内容 | 例 |
|---------|---------|-----|
| 日付セル（勤務あり） | 開始時刻-終了時刻 | `9-17` |
| 日付セル（勤務なし） | ハイフン | `-` |
| 月別合計セル | 合計時間 + アラートアイコン | `32h` `24h ⚠` |

### 2.2 スクロール固定

| 要素 | 固定方向 | 理由 |
|------|---------|------|
| スタッフ名列 | 左固定（sticky left） | 40名スクロール時に誰の行か識別 |
| 日付ヘッダー行 | 上固定（sticky top） | 14日スクロール時に何日か識別 |

### 2.3 ソート機能

| モード | 表示名 | ソートキー |
|--------|--------|-----------|
| `name` | 名前順 | スタッフ名（日本語 `localeCompare`） |
| `totalHours` | 勤務時間順 | 期間内の総勤務時間（降順） |
| `default` | デフォルト | 元の配列順 |

### 2.4 ナビゲーション

| 操作 | 動作 |
|------|------|
| **日付セル押下** | 該当日の1日ビュー（ShiftTableTest）へ遷移 |
| **スタッフ名押下** | スタッフ情報モーダル（StaffEditModal）を表示 |

### 2.5 PDF出力

- **用途**: 印刷・掲示用
- **レイアウト**: A4横 または A3 を想定
- **考慮事項**: 印刷時に見切れないレイアウト設計

### 2.6 アラート機能（別途検討）

| アラート | 条件 | 表示 |
|---------|------|------|
| 週40時間超過 | 週の労働時間 > 40h | 月計セルに ⚠ アイコン |
| 連勤アラート | N日以上連続勤務 | スタッフ行にアイコン |
| 月上限超過 | スタッフの希望上限を超過 | 月計セルにアイコン |

## 5. データ型定義

### 5.1 Props

```typescript
type ShiftOverviewProps = {
  /** シフト期間の開始日 */
  startDate: string;                    // "2026-01-27"
  /** シフト期間の終了日 */
  endDate: string;                      // "2026-02-09"
  /** スタッフ一覧 */
  staffs: StaffType[];
  /** 全シフトデータ（期間内全日分） */
  shifts: ShiftData[];
  /** 祝日リスト */
  holidays?: string[];                  // ["2026-02-11", ...]
  /** スタッフ名クリック時 */
  onStaffClick?: (staffId: string) => void;
  /** 日付セルクリック時 */
  onDateClick?: (date: string) => void;
};
```

### 5.2 スタッフ行データ

```typescript
type StaffRowData = {
  staffId: string;
  staffName: string;
  /** 日付ごとの勤務時間 */
  dailyShifts: Map<string, {           // key: "2026-01-27"
    start: string;                      // "09:00"
    end: string;                        // "17:00"
  } | null>;
  /** 月別合計時間 */
  monthlyTotals: Map<string, number>;   // key: "2026-01", value: 分
  /** アラート情報 */
  alerts: StaffAlert[];
};
```

### 5.3 アラート情報

```typescript
type StaffAlert = {
  type: "week40h" | "consecutive" | "monthLimit";
  /** アラート対象の期間（週の場合は週開始日） */
  targetDate?: string;
  /** 詳細メッセージ */
  message: string;
  /** 実際の値 */
  actualValue: number;
  /** 閾値 */
  threshold: number;
};
```

### 5.4 ソートモード

```typescript
type OverviewSortMode = "default" | "name" | "totalHours";
```

### 5.5 定数

```typescript
const STAFF_NAME_CELL_WIDTH = 120;      // スタッフ名セル幅（px）
const DATE_CELL_WIDTH = 70;             // 日付セル幅（px）
```

## 6. ユーティリティ関数

### 6.1 労働時間計算

| 関数 | 用途 |
|------|------|
| `calculateDailyHours(shift)` | 1日の勤務時間（分）を計算 |
| `calculateMonthlyTotal(shifts, month)` | 月別合計時間を計算 |
| `calculateWeeklyTotal(shifts, weekStartDate)` | 週別合計時間を計算 |
| `getConsecutiveDays(shifts, staffId)` | 連勤日数を計算 |

### 6.2 アラート判定

| 関数 | 用途 |
|------|------|
| `checkWeek40hAlert(shifts, staffId, weekStart)` | 週40h超過判定 |
| `checkConsecutiveAlert(shifts, staffId)` | 連勤アラート判定 |
| `checkMonthLimitAlert(shifts, staffId, limit)` | 月上限超過判定 |
| `collectStaffAlerts(shifts, staffId, config)` | 全アラートを収集 |

### 6.3 日付関連

| 関数 | 用途 |
|------|------|
| `getDateRange(startDate, endDate)` | 期間内の日付配列を取得 |
| `getMonthsInRange(startDate, endDate)` | 期間に含まれる月を取得 |
| `isHoliday(date, holidays)` | 祝日判定 |
| `getDayOfWeek(date)` | 曜日取得 |

## 8. 除外した要件

| 要件 | 除外理由 |
|------|---------|
| ポジションごとの過不足 | 時間帯で変わるため日単位では意味がない。1日ビューで確認 |
| 日ごとの合計人数 | 同上 |
| 日ごとの必要人数 | 同上 |
| 提出状況管理 | 最終チェック用ビューのため不要 |
| フィルター機能 | 初期実装では不要 |
| セル内ポジション表示 | 情報過多。1日ビューで確認 |
| 俯瞰ビュー上での編集 | 役割分担。1日ビューで編集 |
