# スタッフ管理

## 概要

店舗に所属するスタッフの登録・編集・退職処理を行う機能。スタッフはマジックリンクでシフト申請を行い、アカウント登録は不要。

## 関連ファイル

- **Routes**: `src/routes/_auth/shops/$shopId/staffs/`
- **Pages**: `src/components/pages/Staffs/`
- **Features**: `src/components/features/Staff/`
- **Convex**: `convex/staff/`

## 主な機能

- スタッフの新規登録（招待）
- スタッフ情報の編集
- スタッフの退職処理
- スタッフ一覧の表示・検索・フィルタ
- スタッフ詳細の表示
- スキル管理（ポジション × レベル）

## データモデル

```typescript
staffs = {
  // 基本情報
  shopId: Id<"shops">,      // 所属店舗
  email: string,            // メールアドレス
  displayName: string,      // 表示名
  status: string,           // "pending" | "active" | "resigned"
  role: string?,            // "owner" | "manager" | "general"

  // ユーザー紐付け
  userId: Id<"users">?,     // マネージャー判定用

  // スキル・労働条件
  skills: Array<{position, level}>?, // 旧形式（廃止予定）
  maxWeeklyHours: number?,  // 週最大勤務時間

  // マジックリンク
  magicLinkToken: string?,
  magicLinkExpiresAt: number?,

  // 招待トークン
  inviteToken: string?,
  inviteExpiresAt: number?,

  // 管理情報
  invitedBy: string?,       // 招待者authId
  resignedAt: number?,      // 退職日時
  resignationReason: string?, // 退職理由

  // メモ
  memo: string?,            // 管理者用メモ
  workStyleNote: string?,   // AIシフト用
  hourlyWage: number?,      // 時給

  // メタ
  createdAt: number,
  isDeleted: boolean,
}
```

## ステータス遷移

```
pending（招待中）
    ↓ 招待承諾
active（勤務中）
    ↓ 退職処理
resigned（退職済み）
```

## 画面一覧

| 画面 | パス | 説明 |
|------|------|------|
| スタッフ一覧 | `/shops/:shopId/staffs` | スタッフ一覧・検索・フィルタ |
| スタッフ詳細 | `/shops/:shopId/staffs/:staffId` | スタッフ情報・スキル表示 |
| スタッフ編集 | `/shops/:shopId/staffs/:staffId/edit` | スタッフ情報編集 |

## コンポーネント構成

```
Staff/
├── StaffList/      # スタッフ一覧・検索
├── StaffDetail/    # スタッフ詳細・スキル表示
└── StaffEdit/      # スタッフ情報編集
```

## API

### Queries
- `staff.queries.getById` - スタッフ詳細取得
- `staff.queries.listByShop` - 店舗のスタッフ一覧

### Mutations
- `staff.mutations.create` - スタッフ作成（招待）
- `staff.mutations.update` - スタッフ情報更新
- `staff.mutations.resign` - 退職処理
