# スキル管理

## 概要

スタッフごとのスキルレベル（ポジション × レベル）を管理する機能。シフト作成時のスタッフ配置最適化に使用。

## 関連ファイル

- **Features**: `src/components/features/Staff/StaffDetail/`内
- **Convex**: `convex/staffSkill/`

## 主な機能

- スタッフのスキルレベル設定
- ポジションごとのスキル表示
- スキルレベルの更新

## データモデル

```typescript
staffSkills = {
  staffId: Id<"staffs">,        // 対象スタッフ
  positionId: Id<"shopPositions">, // ポジション
  level: string,                // スキルレベル
  updatedAt: number,            // 更新日時
}
```

## スキルレベル

| レベル | 説明 |
|--------|------|
| 未経験 | 業務未経験 |
| 研修中 | 研修・トレーニング中 |
| 一人前 | 独り立ち可能 |
| ベテラン | 熟練者、指導可能 |

## 画面一覧

| 画面 | パス | 説明 |
|------|------|------|
| スタッフ詳細 | `/shops/:shopId/staffs/:staffId` | スキル表示・編集 |

## コンポーネント構成

スタッフ詳細画面（StaffDetail）内でスキル管理UIを提供。

## API

### Queries
- `staffSkill.queries.listByStaff` - スタッフのスキル一覧
- `staffSkill.queries.listByPosition` - ポジション別スキル一覧

### Mutations
- `staffSkill.mutations.upsert` - スキル登録・更新
- `staffSkill.mutations.remove` - スキル削除
