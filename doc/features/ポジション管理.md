# ポジション管理

## 概要

店舗ごとの職種（ポジション）を定義・管理する機能。デフォルトで「ホール」「キッチン」「レジ」「その他」が設定され、店舗ごとにカスタマイズ可能。

## 関連ファイル

- **Features**: `src/components/features/Shop/PositionManager/`, `PositionEditor/`
- **Convex**: `convex/position/`

## 主な機能

- ポジションの追加（最大10個）
- ポジションの編集
- ポジションの削除
- 表示順の変更

## データモデル

```typescript
shopPositions = {
  shopId: Id<"shops">,  // 所属店舗
  name: string,         // ポジション名（最大20文字）
  order: number,        // 表示順
  isDeleted: boolean,   // 削除フラグ
  createdAt: number,    // 作成日時
}
```

## 制約

- ポジション数: 最大10個
- ポジション名: 最大20文字
- デフォルトポジション: `["ホール", "キッチン", "レジ", "その他"]`

## 画面一覧

| 画面 | パス | 説明 |
|------|------|------|
| ポジション管理 | `/shops/:shopId/edit`内 | 店舗編集画面内で管理 |

## コンポーネント構成

```
Shop/
├── PositionManager/   # ポジション一覧・追加・削除
└── PositionEditor/    # ポジション個別編集
```

## API

### Queries
- `position.queries.listByShop` - 店舗のポジション一覧

### Mutations
- `position.mutations.create` - ポジション作成
- `position.mutations.update` - ポジション更新
- `position.mutations.remove` - ポジション削除
- `position.mutations.reorder` - 表示順変更
