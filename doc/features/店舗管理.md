# 店舗管理

## 概要

店舗の登録・編集・削除を行う機能。営業時間やシフト設定を管理する。

## 関連ファイル

- **Routes**: `src/routes/_auth/shops/`
- **Pages**: `src/components/pages/Shops/`
- **Features**: `src/components/features/Shop/`
- **Convex**: `convex/shop/`

## 主な機能

- 店舗の新規登録
- 店舗情報の編集
- 店舗の削除（論理削除）
- 店舗一覧の表示
- 店舗詳細の表示
- 店舗選択（ヘッダーセレクタ）

## データモデル

```typescript
shops = {
  shopName: string,        // 店舗名（2-50文字）
  openTime: string,        // 開店時間（HH:mm形式）
  closeTime: string,       // 閉店時間（HH:mm形式）
  timeUnit: number,        // シフト単位（15/30/60分）
  submitFrequency: string, // 提出頻度（"1w"/"2w"/"1m"）
  avatar: string?,         // アバター画像URL
  description: string?,    // 店舗説明
  createdBy: string,       // 作成者authId
  createdAt: number,       // 作成日時
  isDeleted: boolean,      // 削除フラグ
}
```

## 画面一覧

| 画面 | パス | 説明 |
|------|------|------|
| 店舗一覧 | `/shops` | 所属店舗の一覧表示 |
| 店舗登録 | `/shops/new` | 新規店舗の登録 |
| 店舗詳細 | `/shops/:shopId` | 店舗情報とスタッフ一覧 |
| 店舗編集 | `/shops/:shopId/edit` | 店舗情報の編集 |

## コンポーネント構成

```
Shop/
├── ShopForm/       # 店舗情報フォーム（登録・編集共通）
├── ShopList/       # 店舗一覧カード
├── ShopDetail/     # 店舗詳細表示
├── ShopEdit/       # 店舗編集ページ
├── ShopSelector/   # ヘッダーの店舗選択
├── ShopRegister/   # 新規店舗登録
├── PositionManager/  # ポジション管理（別機能）
├── PositionEditor/   # ポジション編集（別機能）
└── MemberAddModal/   # スタッフ招待（別機能）
```

## API

### Queries
- `shop.queries.getById` - 店舗詳細取得
- `shop.queries.listByUser` - ユーザーの所属店舗一覧

### Mutations
- `shop.mutations.create` - 店舗作成
- `shop.mutations.update` - 店舗更新
- `shop.mutations.remove` - 店舗削除（論理削除）
