# セルフレビュー・チェックリスト

書き終えた直後に必ず1周通す。1問でも「NO」なら直してから返す。

## 共通チェック（どんなテキストでも）

- [ ] 読み手が**一瞬で同じ意味に取れる**か？
- [ ] 具体（数字／動作／固有名）が入っているか？抽象形容詞で逃げていないか？
- [ ] 削れる語がないか？（「こと」「〜することができます」「〜いただきます」連打など）
- [ ] 同じ機能を**別の呼び方**で呼んでいないか？
- [ ] 嘘・誇張・過剰な断定（「完璧に」「必ず」「一切」）がないか？
- [ ] 読んだ人が**次に何をすればいいか**がわかるか？
- [ ] 上から目線・責める語（「不正」「失敗」「間違って」）がないか？

## 日本語の形式チェック

- [ ] 見出し・サブタイトルは**句読点なし**になっているか？
- [ ] 本文は**です・ます**で統一、体言止めの連発になっていないか？
- [ ] 1文が**80字以内**か？長いなら分ける or 箇条書き化
- [ ] 漢字：ひらがなが**3:7**目安か？漢字が多くて堅くなっていないか？
- [ ] 過剰な敬語（「〜させていただく」の連打／二重敬語）が残っていないか？
- [ ] 本文中に**不要な半角スペース**が入っていないか？
- [ ] 数字は**半角算用数字**か？日付表記は統一されているか？
- [ ] カタカナ用語を**日本語に置換**できないか検討したか？
- [ ] 擬音・擬態語（「ぱっと」「サクサク」「スマートに」）を使っていないか？

## LPの場合（追加）

- [ ] ヒーロー（H1）だけ読んで**ベネフィットと対象者**がわかる？
- [ ] H1・サブ・CTAで**同じことを重複**して言っていない？
- [ ] 特徴ではなく**ベネフィット**で書かれているか？
- [ ] 社会的証明（数／ロゴ／声）が**最低1つ**あるか？
- [ ] CTAは「押したあと何が起きるか」を書いている？
- [ ] CTA直下に**不安解消**の一言（クレカ不要／いつでも解約）があるか？
- [ ] 共感セクションは**具体のシーン**になっているか？（抽象悩みで終わっていない）

## プロダクトUI（マイクロコピー）の場合（追加）

### ボタン
- [ ] 動詞始まり・**結果を書いている**か？
- [ ] 破壊的操作は**対象＋動作**が両方入っている？（「このシフトを削除」）

### フォーム
- [ ] ラベルは**枠の外・上**にあるか？
- [ ] プレースホルダーにラベルを入れていない？入れるなら**入力例のみ**か？
- [ ] 必須／任意の表記ルールは統一されているか？

### エラーメッセージ
- [ ] 「何が起きたか」と**「どうすれば直るか」**が両方書かれているか？
- [ ] 責める語（「不正」「失敗」）を使っていない？
- [ ] エラー発生場所の**近く**に出ているか？

### 空状態
- [ ] この画面で**何ができるか**が書かれているか？
- [ ] **最初の一歩**（アクションボタン）が示されているか？

### 成功トースト
- [ ] ドヤっていない？（「ばっちり！」「完璧！」はNG）
- [ ] Undo が必要な操作にボタンが添えられているか？

## 小規模店舗・個人事業主向けの追加チェック

- [ ] カタカナ用語を**日常語**に置き換えたか？（ダッシュボード→画面／アラート→お知らせ）
- [ ] 業務動作の**具体名**が入っているか？（シフト表／一斉送信／レジ連携）
- [ ] 「小さな店」「零細」などの見下し表現を避けているか？
- [ ] 「DX／イノベーション／最適化」が実感のある言葉に置き換わっているか？
- [ ] 「AI／自動で／勝手に」の**過剰な期待**を作っていないか？

## AI-slop 警戒リスト（見つけたら必ず書き直す）

以下の語・表現を自分の出力の中で見つけたら、ほぼ書き直しが必要。

- 「革新的な」「シームレスな」「最先端の」「最高水準の」「業界をリードする」
- 「パワフルな」「スマートな」「プロフェッショナルな」「強力な」
- 「効率化」「最適化」「生産性向上」「イノベーション」（抽象すぎ）
- 「ぱっと」「サクサク」「スムーズに」（擬音・擬態語）
- 「〜することができます」（→「〜できます」）
- 「〜させていただきます」の連打
- 「ご利用いただけますと幸いです」（→「使ってください」）
- 「自信を持ってお届け」「寄り添い続けて」
- 「DX」「ソリューション」「ワークフロー」（使わない理由があるかで判断）

## 最終関門

1. **声に出して読む**：一度も詰まらずに読めるか？
2. **説明を削る**：2割削ってみる。意味が壊れなければ元に戻さない
3. **5秒ルール**：読み手は5秒後に要点を1行で言えるか？
4. **誰向けか**：読み手の顔が思い浮かぶか？（ぼんやりした"ユーザー"ではなく、具体の人）

1つでも自信がなければ、書き直す。迷ったときは **短く・具体的・肯定形** が正解の方向。
