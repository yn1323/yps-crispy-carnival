# LINEでシフト希望を集めると提出が流れるときの対策

グループLINEでは、他の連絡が重なるとシフト希望が見つけにくくなることがあります。

## 起きやすいこと

- 提出依頼の投稿が流れる
- 変更連絡が途中に混ざる
- 誰が出したか確認し直す必要がある

## 対策の考え方

提出依頼の文面を固定し、締切前に未提出者だけを確認できるようにしておくと、探す時間を減らせます。

## まとめ

LINEで集める場合も、提出場所と確認方法を決めることが大切です。
