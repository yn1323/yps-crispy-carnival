# LINEで使えるシフト希望提出依頼の文面例

LINEで依頼するときは、短くても必要な情報がそろっている文面にすると、スタッフが動きやすくなります。

## 入れておきたい内容

- 対象期間
- 提出期限
- 希望の書き方
- 提出先

## 文面例

「来月分のシフト希望を〇日までに送ってください。勤務できる日と時間、休み希望があれば一緒にお願いします。」

## 迷わせない工夫

毎月同じ型で依頼すると、スタッフも提出方法を覚えやすくなります。
