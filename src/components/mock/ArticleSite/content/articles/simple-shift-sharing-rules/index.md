# 確定シフトをスタッフに共有するときの注意点

確定シフトを共有するときは、どこを見れば最新なのかが分かる状態にしておくと見落としを減らせます。

## 共有前に確認すること

| 項目 | 例 |
| --- | --- |
| 対象期間 | 何月分のシフトかを書く |
| 確認場所 | 最新版を見る場所を固定する |
| 変更連絡 | 確定後の変更先を決める |

## 変更連絡を分ける

確定前の希望変更と、確定後の勤務変更は扱いを分けると、担当者も確認しやすくなります。

## まとめ

共有先と変更ルールを固定するだけでも、スタッフが迷う場面は減らせます。
