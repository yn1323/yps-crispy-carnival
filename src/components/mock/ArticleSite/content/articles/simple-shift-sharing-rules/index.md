# スタッフが希望を出しやすくなるシンプルなルール例

シフト希望の出し方が人によって違うと、集計や確認に時間がかかります。最初から細かくしすぎず、最低限のルールを共有するのが続けやすい方法です。

## ルール例

| 項目 | 例 |
| --- | --- |
| 提出期限 | 毎月20日まで |
| 書き方 | 勤務できる日と時間を書く |
| 変更連絡 | 確定前は提出先へ、確定後は店長へ連絡 |

## 大切なのは迷わないこと

スタッフにとって分かりやすいルールは、担当者にとっても確認しやすいルールです。提出方法と確認場所を固定するだけでも、やり取りは少なくなります。
