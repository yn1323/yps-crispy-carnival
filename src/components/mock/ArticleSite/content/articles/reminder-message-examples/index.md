# シフト希望のリマインド文面例

リマインドは、強く催促するためではなく、提出を忘れているスタッフが動きやすくなるように送るものです。

## 伝える内容を短くする

- 対象期間
- 提出期限
- 提出先

この3つを短くまとめるだけで、スタッフは次に何をすればよいか分かりやすくなります。

## タイミングを決める

締切前日だけでなく、募集開始時と締切前の2回に分けると、忘れにくくなります。
