# シフト希望の未提出者を減らすには？

未提出の人を毎回探す作業は、シフト作成の前に起きる小さな負担です。提出済みと未提出が分かれていないと、確認に時間がかかります。

## まず提出状況を見えるようにする

- 誰が提出済みか
- 誰が未提出か
- いつまでに提出してほしいか

この3つが見えるだけで、声をかける相手を迷いにくくなります。

## 全員に催促しない

すでに提出したスタッフにも何度も通知すると、次から読まれにくくなることがあります。未提出者だけに送れる状態にしておくと、担当者もスタッフもラクです。

## 締切を固定する

毎月の締切日が変わりすぎると、スタッフが提出タイミングを覚えにくくなります。提出依頼と締切をセットで伝えることが大切です。
