# Excelでシフト表を作るのが大変になる理由

Excelは最初のシフト表作成には便利です。自由に表を作れて、印刷もしやすく、すでに使い慣れている人も多いからです。

ただし、希望回収、未提出確認、確定後の共有まで含めると、Excelの外側に手作業が残りやすくなります。

## 大変になりやすいこと

| 困りごと | 起きやすい理由 |
| --- | --- |
| 希望を転記する | LINEや紙で集めた内容を表に写す必要がある |
| 最新版が分からない | ファイル名や共有先が増える |
| 変更が伝わらない | 確定後の連絡が別の場所で行われる |

## Excelが合っているケース

- スタッフ数が少ない
- 固定シフトが多い
- 変更がほとんどない
- シフト担当者が一人で管理できる

この条件に近ければ、Excelのままでも十分なことがあります。無理にツールを変えるより、入力ルールを整える方が早い場合もあります。

## 見直しどころ

毎月、希望を写す、未提出者を確認する、最新版を送り直す。この作業が続いているなら、表そのものではなく、希望回収から共有までの流れを見直すタイミングです。
